### Installation

```sh
$ git clone git@github.com:marryday/grp-ib-test.git 
$ cd grp-ib-test/api/ 
$ mv .env.example .env
$ npm i
$ npm start
$ cd ../client
$ npm i 
$ npm start  ||  yarn start
```

For production environments...

```sh
$ npm install --production
$ NODE_ENV=production node app
```
